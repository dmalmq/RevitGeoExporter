Export Dialog

エクスポートの流れ

新しいエクスポート画面では、標準的な操作を先に見せ、技術的な設定は展開式にまとめています。初めてのユーザーでも、通常のエクスポートなら迷わず進める構成です。

Export to Include in export Coordinate system Advanced options

Export to

  • 出力フォルダーは必須項目です。ダイアログ上部で早い段階に確認できます。
  • Browse... を使って保存先を変更できます。
  • ビュー、フィーチャータイプ、出力先がそろうまで Export は有効になりません。

Include in export

  • Units、Details、Openings、Levels を選択します。
  • 画面上のラベルは分かりやすい表現ですが、GeoPackage には unit や opening など既定のレイヤー名で出力されます。
  • 少なくとも 1 つのビューと 1 つのフィーチャータイプを選ぶと、Preview を開けます。

Coordinate system

  • 状態カードに、現在の共有座標と解決された EPSG が要約表示されます。
  • 共有座標ではなく変換出力が必要な場合だけ、Change coordinate settings を開いて CRS や EPSG を変更します。
  • より詳細なモデル座標の情報が必要なときは、View technical details を展開します。

Advanced options

  • Advanced options は初期状態で折りたたまれており、主な作業を邪魔しません。
  • Unit source は Units を有効にしたときだけ表示され、Rooms を選んだ場合のみ部屋カテゴリパラメータが表示されます。
  • プロジェクトマッピング、診断レポート、GIS パッケージ、凡例ファイルの設定も、必要なときにだけ確認できます。

書き出し前の確認

フッターには現在の選択内容が要約表示され、実際の書き出し前には検証が実行されます。

  • ジオメトリや警告、カテゴリ割り当てを確認したい場合は プレビューと割り当て を開きます。
  • どのような項目がエクスポートを止めるかは 検証と診断 を参照してください。
