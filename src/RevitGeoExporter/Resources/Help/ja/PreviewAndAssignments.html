プレビューと割り当て

プレビュー画面もエクスポート画面と同じカード型レイアウトになりました。左にフィルター、中央にマップ、右にインスペクターを配置しています。

Search Layers Details Assignments Warnings

画面構成

  • 左側には Search、Map layers、Filters、View tools があります。
  • 中央には凡例、状態表示、プレビューキャンバスがあります。
  • 右側のインスペクターに Details、Assignments、Warnings がまとまっています。

フィーチャーの確認

  • キャンバス上のフィーチャーをクリックすると、Details タブにメタデータが表示されます。
  • Warnings タブでは、そのビューに関係する検証・準備時の問題を確認できます。
  • 何も選択していないときは、次に何をすべきかが分かる空状態メッセージが表示されます。

割り当て

  • カテゴリが未確定の床由来ユニットに対して、Revit モデルを変更せずにエクスポーター側のカテゴリを割り当てられます。
  • Assignments タブには、解析候補、現在値、保留中の変更がまとめて表示されます。
  • 割り当て、上書き解除、保存、破棄を行ってからプレビューを閉じられます。

フィルターと表示操作

  • Search では名前、カテゴリ、エクスポート ID で絞り込みできます。
  • レイヤー切り替え、警告のみ、上書きのみ、未割り当てのみの各フィルターはすぐにマップへ反映されます。
  • ベースマップや測量点を表示できない場合は、無効化されたまま理由が画面内に表示されます。
  • Fit と Reset で表示を整え、マウスホイールのズームやドラッグ移動も継続して使えます。
