Getting Started

RevitGeoExporter is a Revit 2024 add-in for exporting selected plan views to IMDF-style GeoPackages and previewing the result before writing files.

Before you export

  • Open a Revit project that contains exportable plan views.
  • Confirm the default output folder and CRS from the settings entry inside Export GeoPackage.
  • Use the export dialog to choose views and feature layers.

Main workflow

  1. Open Export Workflow from the help viewer or click Export GeoPackage on the ribbon.
  2. Preview units, openings, details, and levels before export when needed.
  3. Review validation results and diagnostics warnings before writing files.
