Export Workflow

The export dialog is the main place for selecting views, feature layers, CRS, package output, and geometry repair options.

Typical flow

  1. Select one or more plan views.
  2. Choose the feature layers to export: unit, detail, opening, and level.
  3. Review output folder, target EPSG, diagnostics, packaging, and repair settings.
  4. Use Preview if you want to inspect geometry before export.
  5. Run export and review the validation dialog.
  6. After export, review files, warnings, package output, and change summaries.

Notes

  • Profiles can save export options for reuse.
  • Package output can include preview images, a manifest, and diagnostics.
  • Validation runs before writing files and can block export on hard errors.

See Preview and Assignments for preview-specific behavior.
