Settings and Profiles

The settings hub separates Global defaults from Project mappings and scoped data.

Global

  • Default output folder
  • UI language
  • CRS / EPSG
  • Diagnostics, packaging, and repair defaults
  • Global export profiles

Project

  • Floor type to category mappings
  • Family category overrides
  • Accepted opening family names
  • Project-scoped export profiles

Import, export, and reset

Each scope can be imported, exported, or reset independently. Malformed saved files appear in the settings status area and safe defaults are used when needed.
