Troubleshooting / FAQ

Why is something missing in QGIS?

Use the preview window and validation dialog first. Missing vertical units, unsnapped openings, and unsupported families should appear there before export completes.

Why did a floor export as unspecified?

The floor type name did not resolve to a known category. Use the preview assignment workflow or project mappings to add an exporter-side override.

Where are settings stored?

Global defaults and project mappings are stored in the RevitGeoExporter application data area. The settings hub can import, export, and reset each scope.

Can the add-in modify the Revit model?

Preview mappings and help content are exporter-side only. They do not rename families, rename floor types, or write category decisions back into the Revit model.
