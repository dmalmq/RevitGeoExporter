Validation and Diagnostics

The exporter validates prepared data before writing files and can emit a diagnostics report beside the GeoPackage output.

Validation checks include

  • Unassigned floor-derived units
  • Duplicate or missing export IDs where required
  • Empty or invalid geometry
  • Unsupported opening families and unsnapped openings
  • Views with no exportable output
  • Missing vertical circulation output

Diagnostics output

The diagnostics JSON captures view counts, warnings, skipped or unsupported content, overrides, timestamps, and export timing. Package output can include the diagnostics report in the handoff folder.
