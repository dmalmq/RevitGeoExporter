Troubleshooting / FAQ

Use this page when export behavior is technically correct but still unexpected for the current project or receiving workflow.

Why is something missing in QGIS?

Start with Preview and Assignments and the validation dialog. Missing vertical units, unsnapped openings, unsupported families, and filtered-out layers usually appear there before export completes.

Why did a floor export as unspecified?

The floor type name did not resolve to a known category. Use preview assignments or the project mappings area in Settings and Profiles to add an exporter-side override.

Where are settings stored?

Global defaults and project mappings are stored in the RevitGeoExporter application data area. The settings hub can import, export, and reset each scope independently.

Can the add-in modify the Revit model?

No. Preview assignments, mappings, diagnostics, and help content are exporter-side only. They do not rename Revit elements or write category decisions back into the model.

Still blocked?

  • Review Validation and Diagnostics for issues that can stop the export.
  • Review Export Workflow to confirm the selected layers, folder, and coordinate mode.
