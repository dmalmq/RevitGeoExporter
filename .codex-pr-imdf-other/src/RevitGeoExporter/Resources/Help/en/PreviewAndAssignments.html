Preview Workspace

Preview and Assignments

The preview window now mirrors the export dialog with a card-based layout: filters on the left, the map workspace in the center, and the inspector on the right.

Search Layers Details Assignments Warnings

Layout overview

  • The left rail contains Search, Map layers, Filters, and View tools.
  • The center workspace shows the legend, status hints, and the preview canvas.
  • The right inspector keeps Details, Assignments, and Warnings together in one place.

Inspecting features

  • Click a feature on the canvas to inspect its metadata in the Details tab.
  • The warnings tab groups validation or preparation issues that are relevant to the current view.
  • An empty-state message is shown when nothing is selected, so the next step is obvious for new users.

Assignments

  • Unassigned floor-derived units can be reviewed and assigned to supported categories without modifying the Revit model.
  • The assignments tab shows the parsed candidate, the current value, and any pending change before you save.
  • You can assign, clear an override, save staged changes, or discard them before closing the preview.

Filters and view tools

  • Search filters by feature name, category, or export ID.
  • Layer toggles, warning filters, override filters, and unassigned-only filters update the map immediately.
  • Basemap and survey-point options stay disabled with an inline reason when the current preview cannot show them.
  • Use Fit and Reset for quick framing, and keep using mouse wheel zoom and drag-to-pan navigation.
