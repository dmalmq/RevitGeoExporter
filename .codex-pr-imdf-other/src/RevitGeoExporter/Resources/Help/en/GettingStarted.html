Quick Start

Getting Started

RevitGeoExporter now guides most users through a simple path: select plan views, choose an output folder, choose what to include, preview when needed, and export.

Plan views Output folder Preview Validation

Before you export

  • Open a Revit model that contains the floor or ceiling plan views you want to publish.
  • Review project mappings or accepted opening families in Settings and Profiles if the project uses custom naming.
  • Confirm that the output folder and coordinate assumptions match the needs of the receiving GIS team.

Guided workflow

  1. In Plan views, select one or more views to export.
  2. In Export to, confirm the folder or use Browse... to choose a new one.
  3. In Include in export, choose the layers you want to write: unit, detail, opening, and level.
  4. In Coordinate system, keep shared coordinates for the default path or expand the settings if you need to convert to a target CRS or EPSG.
  5. Use Preview to inspect geometry, warnings, and floor-category assignments before files are written.
  6. Run Export, resolve any blocking validation issues, and then review the results summary.
