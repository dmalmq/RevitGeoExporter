Quick Start

はじめに

RevitGeoExporter は、まず標準的な流れを分かりやすく見せるようになりました。プランビューを選び、出力先を決め、含める内容を選び、必要に応じてプレビューしてからエクスポートします。

プランビュー 出力フォルダー プレビュー 検証

エクスポート前に確認すること

  • 出力したい平面図または天井伏図を含む Revit モデルを開きます。
  • 独自の命名や例外的な開口ファミリがある場合は、設定とプロファイルでプロジェクト側のマッピングを確認します。
  • 出力フォルダーと座標の前提が、受け渡し先の GIS ワークフローに合っていることを確認します。

基本の流れ

  1. Plan views で、エクスポートするビューを 1 つ以上選択します。
  2. Export to で出力フォルダーを確認し、必要なら Browse... で変更します。
  3. Include in export で、unit、detail、opening、level の各レイヤーを選びます。
  4. Coordinate system では通常は共有座標のままで進め、必要なときだけ設定を展開して CRS や EPSG 変換を指定します。
  5. ファイルを書き出す前に、Preview でジオメトリ、警告、カテゴリ割り当てを確認できます。
  6. Export を実行し、検証で止まった項目を解決したあと、結果画面で出力内容を確認します。
